# `opuncleh setup`

Initialize `~/.opuncleh/opuncleh.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
opuncleh setup
opuncleh setup --workspace ~/.opuncleh/workspace
```

To run onboarding via setup:

```bash
opuncleh setup --wizard
```
